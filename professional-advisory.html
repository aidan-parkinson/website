Systems Transformation: BUS Methodology Partner Network

The BUS Methodology Partner network consists of ~35 licensed partners who are provided with the training and resources to deliver occupant satisfaction evaluation and benchmark analysis using the BUS Methodology tool.

I have taken a leading role in a transformation of the service to automate processes, enhance customer experience and deliver the database of >70000 consistent response records in a way that should realise more of the products potential.

I specified and provided content for: a static HTML marketing website; a React web user-interface; an EVE REST API; and a MongoDB Atlas database cluster. Deployment of this domain is somewhat automated with Terraform and employs various AWS services (S3, Elasticbeanstalk, Secrets Manager, Cloudfront, Web Application Firewall, Route 53, IAM, Certificates Manager).

I managed the deployment through a Beta testing phase with 3rd party penetration tests and now maintain it, carrying out functional testing and making changes to both software and deployment configurations as required. All domain services have now achieved general customer release.

Systems Integration: Google KGX1

During 2019 I reviewed the majority of technical submittals by specialist contractors for compliance with system integration requirements specified by others.

I provided assurance through witnessing test functionality of a software tool under development for qualifying network edge devices for enrollment on a converged enterprise IP network. The DAQ (Device Automated Qualification) software is designed for continuous deployment as a package of Docker containers running on a Faucet compatible Openflow switch controller. The code is managed in an open-source repository.
